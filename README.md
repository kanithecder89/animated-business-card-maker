# Animated Business Card Maker 

A web app to create animated and customizable digital business cards with live editing.

##  Features
- Live text, color, and logo editing
- Smooth animations
- QR code integration
- Download custom business cards

##  Tech Stack
HTML5, CSS3, JavaScript

##  Purpose
Demonstrates creativity with JavaScript animations and DOM manipulation.
